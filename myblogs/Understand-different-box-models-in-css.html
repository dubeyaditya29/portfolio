CSS box model is a container is used to create the design and layout of web pages. Imagine everything being rendered on the screen as a BOX, which has some properties that helps to modify the design.

The above picture is from the Chrome Development tool. Just use (control + shift +i)

Elements of the box -

  • Margin
  • Padding
  • Border
  • Content

-- The Margin --

The margin is the exterior section. You can consider this as an area that separates the interior of the CSS box model from other HTML elements on a page. Read More about Margins

-- The Padding --

Padding can be considered as extra coushions for the BOX.It sits in the space between the HTML content and the border.You can add padding on left,right,bottom and top it completely depends on the requirement Read More about padding

-- The Border --

The Border is a layer between margin and padding. A border can be used for multiple things..we can give it a style like dotted,dashed,solid and also we can set differnet colors..with border-radius property we can give curved corners to the borders..read more about border radius. Read More about Borders

-- The Content --

The content is the inner most part of the box model and this is where you actual content appears on the screen..it can be anything Text,Image or any media content Read More about Contents
